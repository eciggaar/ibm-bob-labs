# Getting Started

Welcome! This page shows how your markdown content will be rendered.

## Prerequisites

- Node.js 18 or later
- A GitHub account

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/your-org/your-repo.git
cd your-repo
npm install
```

## Running locally

```bash
npm run dev
```

Open [http://localhost:4321](http://localhost:4321) in your browser.

## Next steps

- Read the [Configuration guide](/docs/configuration) to customise the site.
- Add your own `.md` files to `src/content/docs/`.
